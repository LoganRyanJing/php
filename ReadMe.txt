增加的需求：

一、增加成绩分析功能

	1.统计分析考试结果，生成统计分析“区域合格率”的图表，饼状图显示；

	2.统计每场考试平均分，对各个区域（店铺）之间的平均分进行比较；

	3.能够自动生成 不同分数段之间的占比，要有饼状图就行显示。

二、增加区域划分

	增加区域划分功能，要添加不同区域，对区域进行管理。用户信息中包含区域信息，注册时添加。

三、导出成绩

	考试结果：可导出EXCLE的成绩表（包括考试时间、姓名、工号、门店等信息）；可删除
	
四、增加店铺管理

	增加店铺管理功能，要添加不同店铺，对店铺进行管理。用户信息中包含店铺信息，注册时添加。

五、更改界面

	修改注册页面，注册添加姓名、工号、区域、门店等信息，要修改考试界面，考试信息已列表形式展示，要有考试提醒。
	
六、增加文件系统

	对系统文件进行管理，文件上传，文件下载功能，增加例如ppt等格式的下载功能，能够看培训资料的模块。
